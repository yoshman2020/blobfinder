# BlobFinder

BlobFinder は、画像処理およびブロブ解析をブラウザ上で実行できる Web アプリケーションです。

画像の読み込み、表示操作、各種画像処理、ブロブ解析、および解析結果のエクスポート機能を提供します。

---

# 主な機能

## 画像表示

* 画像ファイルの読み込み
* 画像の表示
* 画像の拡大・縮小
* 画像表示位置の移動（パン操作）
* 元画像サイズでの表示
* ウィンドウサイズへのフィット表示
* 元画像と処理後画像の表示切り替え

## 画像処理

複数の画像処理を組み合わせて適用できます。

* 複数処理の選択
* 処理順序の変更
* 各処理パラメーターの設定
* 処理結果のリアルタイム表示
* 処理結果画像の保存

### 色空間・チャンネル処理

* R抽出
* G抽出
* B抽出
* Gray変換
* H抽出
* L抽出
* S抽出

### コントラスト補正

* ヒストグラム平滑化
* 適応ヒストグラム平滑化（CLAHE）

### 平滑化フィルタ

* 平滑化（Blur）
* メディアンフィルタ
* バイラテラルフィルタ

### 二値化

* 二値化
* 適応二値化

### モルフォロジー処理

* オープニング
* クロージング
* 膨張（Dilation）
* 縮小（Erosion）
* モフォロジー勾配
* トップハット
* ブラックハット

### エッジ検出

* Sobel
* Laplacian
* Scharr
* Canny

### 領域処理

* 穴埋め
* 境界ブロブ除去
* ウォーターシェッド

### 解析

* ブロブ解析

---

# ブロブ解析機能

ブロブ解析結果を一覧表示できます。

## 表示項目

* 重心 X 座標
* 重心 Y 座標
* 面積
* 真円度
* 凸度
* 慣性比
* 周囲長
* 回転を考慮した外接矩形の長辺長
* 回転を考慮した外接矩形の短辺長

## 一覧機能

* 各列でソート可能
* 行選択による対象ブロブのハイライト表示
* CSV形式での保存

---

# ファイル管理

サーバー上に不要な画像ファイルが蓄積されないよう、アップロード画像および処理結果画像は適切に管理されます。

* 一時ファイルの自動削除
* 不要ファイルの定期クリーンアップ
* 永続保存を行わない設計

# 起動方法

python3 -m uvicorn app:app --host 0.0.0.0 --port 8000

ブラウザで
http://localhost:8000

# 操作手順

1. 画像ファイルを読み込む
2. 画像表示を拡大・縮小・移動して確認する
3. 必要な画像処理を選択する
4. 各処理のパラメーターを設定する
5. 処理順序を調整する
6. 処理結果を確認する
7. 必要に応じて画像を保存する
8. ブロブ解析を実行する
9. 解析結果を確認し、CSV出力する
